# Installation

Install from PyPI:

```bash
pip install splink-snowflake
```

Or with [uv](https://github.com/astral-sh/uv):

```bash
uv add splink-snowflake
```

This will also install `splink` and `snowflake-connector-python[pandas]` as dependencies.

## Requirements

- Python >= 3.12
- A Snowflake account
