# `splink-snowflake`

[Snowflake](https://www.snowflake.com) support for use as a backend with the data-linkage and deduplication package [Splink](https://moj-analytical-services.github.io/splink/).

> [!WARNING]
> The Splink Snowflake backend is early stage, and not fully stable or tested at time of writing.
